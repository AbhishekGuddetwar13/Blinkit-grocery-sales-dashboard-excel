# Blinkit Grocery Sales Dashboard (Excel)

## Project Overview

This project is an interactive Excel dashboard built using Blinkit grocery sales data. The dashboard provides insights into sales performance, item categories, outlet types, customer ratings, and sales trends.

The dashboard was developed using Microsoft Excel with Pivot Tables, Pivot Charts, Slicers, and Dashboard Design techniques to transform raw data into meaningful business insights.

## Business Insights

### Top Performing Categories
Snack Foods and Fruits & Vegetables contributed significantly to total sales.

### Outlet Performance
Supermarket outlets generated a major share of sales compared to other outlet types.

### Customer Satisfaction
The average customer rating remained at 3.92, indicating positive customer feedback.

### Sales Trend Analysis
Sales showed consistent growth over the years, with peak performance observed in later periods.
